Emoji Math
Click on the start button to see a Target Number. Click on an emoji to add to the User Number.
Continue clicking on the emojis until you are able to get your User Number to equal the Target Number.
If you can get your number to match, you win! If you go over the target number, you lose.
Each time you play a new random number is selected as your Target Number and the value of each emoji changes.
